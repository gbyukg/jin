jin
===

A responsive Octopress theme, inspired by Evernote and Octopress classic theme.

##How to install

```
$ cd your_octopress_directory
$ git clone https://github.com/lijinma/jin.git .themes/jin
$ rake install['jin']
$ rake generate
```
